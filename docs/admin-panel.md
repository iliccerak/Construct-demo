# Platform Admin Panel Structure

## Core Sections

1. **Global Dashboard**
   - Platform KPIs
   - Active companies
   - Revenue summaries

2. **User Management**
   - Search, suspend, reactivate
   - MFA reset
   - Audit trail per user

3. **Company Oversight**
   - Subscription tier management
   - Compliance status
   - Legal details verification

4. **Subscription & Billing**
   - Manual bank payment verification
   - Invoice dispute tracking

5. **Security & Abuse Monitoring**
   - IP restriction rules
   - Suspicious login alerts
   - Abuse and fraud reports

6. **System Configuration**
   - Feature toggles
   - Environment variables (masked)
   - Email templates

7. **Audit Logs**
   - Global audit log search
   - Export for compliance
